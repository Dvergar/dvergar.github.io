API Documentation

class NetEntityManager

package osis

extends

import osis.EntityManager

Available on all platforms

Available via EntityManager.listen on server or EntityManager.connect on client.

NetEntityManager is basically mirroring EntityManager but each call will be done locally AND through the network.

Methods

addComponent<T> (entity:Entity, component:T):T

Available on server

Add an component locally AND on the network.

addComponentTo<T> (entity:Entity, component:T, connEntity:Entity):T

Available on server

Add an component locally AND on the network at the specified connection linked to connEntity.

addEvent<T> (messageClass:Class<Message>, func:T ‑> Connection ‑> Void):Void

Link an event messageClass to a func.

Usage :

class MessageHello implements Message
{
    @String public var txt:String;

    public function new() {}
}

netEntityManager.addEvent(MessageHello, function(msg:MessageHello, connection:Connection) {
    trace(msg.txt);
});

bindEntity (connection:Connection, entity:Entity):Void

Available on server

Links connection to entity.
The linked entity will be destroyed on disconnection.

createEntity (name:String):Entity

Available on server

Create an entity of template name locally AND on the network.

destroyEntity (entity:Entity):Void

Available on server

Destroy an entity locally AND on the network.

getBoundEntity (connection:Connection):Null<Entity>

Available on server

Get entity registered via bindEntity.

removeComponent<T> (entity:Entity, componentType:T):Void

Available on server

Remove a component locally AND on the network.

sendEvent (message:Message, ?connection:Connection):Void

Send event message on the network to everyone or only to connection if specified.

sendWorldStateTo (connection:Connection):Void

Available on server

Send all the server entities to connection.
