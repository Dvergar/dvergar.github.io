QOOKIE

OpenFL abstraction for the flash.net.Socket class.

A simple library adding a few helpers around the OpenFL ByteArray class to recreate the flash.net.Socket behaviour.

SISO

Entity Component System architecture with networking support

  • Automatic & fine-grained serialization of components via PODStream
  • API support for shared code betwen client & server

Per variable serialization with @short @byte... metadatas.

Iteration on another library of mine ENH.

Language: Haxe Source

Anette

Anette is a haxe network library with a simple API.

Supports C++, Neko, Flash and Javascript client-side & C++, Neko and Javascript (Node) server-side.

Handles connections/disconnections & supports custom protocols.

Language: Haxe Source
