After my first steps in python i decided to invest my time into game development. I made many multiplayer online prototypes, participated into game jams and worked on many games. I set on minimalistic puzzle games, sometimes targeted at children. I keep only the raw mechanic in my games.

Tech

I've been developing games in haxe, javascript, Lua, Python and in game engines such as Unity and Godot Engine but my expertise is in the Haxe language, as i can target any platform and makes porting as3 (flash) games to web or desktop fast.

Games
